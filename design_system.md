# Design System - OnRadio Portal

Este documento define las reglas de diseño y componentes para el portal público de radio.

## Paleta de Colores
- **Primario**: Definido por el tenant (Ej: #ff0000)
- **Secundario**: Definido por el tenant (Ej: #000000)
- **Fondo**: Gradientes suaves o colores oscuros premium.

## Tipografía
- Inter (Google Fonts)
- Roboto (Google Fonts)
- Outfit (Google Fonts)

## Componentes UI
- **GlobalPlayer**: Reproductor persistente en la parte inferior o fija.
- **PublicNavbar**: Navegación principal con logo y búsqueda.
- **PollWidget**: Widget de encuestas interactivas.
- **AdsWidget**: Espacios publicitarios en Sidebar y Main.
- **PodcastCard**: Tarjetas para visualización de episodios.

## Estándares de Diseño
- Glassmorphism para modales y widgets.
- Micro-animaciones con `framer-motion` o CSS transitions.
- Responsive design priorizando mobile-first.
